## ✨ Why RTX Edition?

기존의 Wiggle 2를 넘어선 강력한 기능들을 경험해 보세요:

*   **Modern API Support**: Blender 4.2 LTS, 5.0 및 5.1+ 환경을 완벽하게 지원합니다.
*   **Layered Physics System**: 부위별로 독립적인 물리 제어가 가능한 혁신적인 레이어 시스템.
*   **Keyframe & Sim Baking**: 시뮬레이션 데이터를 애니메이션 키프레임으로 변환하여 게임 엔진 호환성을 극대화합니다.
*   **High-Density Optimization**: 수만 개의 본(Bone)이 얽힌 복잡한 리그에서도 뷰포트 렉 없이 부드럽게 작동합니다.
